# Developer Images

Place developer profile images in this folder with the following names:

- `abhishek.jpg` - Abhishek Pandey (Frontend Developer)
- `akashdeep.jpg` - Akashdeep (Backend Developer)
- `gagan.jpg` - Gagan Kumar Shah (Database Developer)

## Image Requirements:
- Format: JPG, PNG, or WebP
- Recommended size: 400x400 pixels (square)
- File size: Keep under 500KB for optimal loading

## Fallback:
If images are not found, the component will automatically generate placeholder avatars with the developer's initials.
